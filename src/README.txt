This directory contains all the source needed to build the uPerm interpreter.

Build instructions:
1) Download and install ANTLR v2.3 from www.antlr.org.
2) Add the antlr.jar to your classpath, read the ANTLR website to learn how to...
3) Use ANTLR to convert uperm.g into a Lexer and Walker.
4) Build the generated files and all uPerm Java classes contained in this directory.
5) Jar it all up into uperm.jar (don't forget Manifext.txt).
6) Done!
